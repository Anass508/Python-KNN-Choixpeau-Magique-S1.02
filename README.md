🧙‍♂️ Le Choixpeau Magique 2.0 - Classification k-NN📌 Présentation du ProjetCe projet a été réalisé dans le cadre de la SAÉ S1.02 (2025-2026). Il s'agit de développer une méthode de répartition des élèves de Poudlard plus performante qu'un simple choix aléatoire. Le système utilise l'informatique pour analyser les traits de personnalité des élèves et les affecter à la maison la plus proche.⚙️ FonctionnalitésLe projet implémente les concepts suivants :Modélisation des données : Stockage des élèves dans des dictionnaires où les clés sont les noms et les valeurs sont des tableaux de 10 entiers (réponses).Distance Euclidienne : Calcul de la similarité entre deux profils dans un espace de dimension 10.Algorithme k-NN (k plus proches voisins) : Classification basée sur les $k$ références les plus proches pour une décision plus robuste.Tri par insertion optimisé : Gestion dynamique d'une liste triée de voisins pour maintenir une complexité efficace.📂 Structure du Dépôts101.py : Correction de la première partie du projet.s102.py : Code principal contenant les fonctions de classification (k-NN) et de calcul de distance.test_s102.py : Suite de tests unitaires pour valider chaque fonction.conclusion_question_sae101.ipynb : Analyse expérimentale et comparaison des taux d'erreur.data/ : Contient les fichiers de référence des fondateurs (houses_ref.json) et les 40 magiciens illustres (houses_multiple_refs.json).🚀 UtilisationPour tester le code et vérifier la précision de l'algorithme :Assurez-vous d'avoir Python installé.Exécutez les tests unitaires :Bashpython test_s102.py
📊 Évaluation techniqueLe projet respecte les critères suivants :Tests unitaires : Validation automatique du code.Qualité du code : Code commenté, utilisation de docstrings et respect de la clarté algorithmique.Analyse : Comparaison des pourcentages d'erreurs pour $k$ allant de 1 à 5.
